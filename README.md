# 🩺 Diabetes Prediction using Artificial Neural Networks

A Deep Learning project that predicts whether a patient is diabetic based on medical attributes using an Artificial Neural Network (ANN).

This project demonstrates the complete machine learning workflow, including data preprocessing, model building, training, evaluation, and visualization using TensorFlow/Keras.

---

## 📖 Project Overview

Early detection of diabetes can help improve patient care and reduce long-term health risks.

In this project, I built an Artificial Neural Network that learns patterns from patient health data to predict the likelihood of diabetes.

The project covers every stage of the machine learning pipeline—from data preprocessing to model evaluation and visualization.
